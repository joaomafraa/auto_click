# Autocliquer

App simples em Python para abrir um navegador controlado, gravar cliques e repetir a sequencia ate pressionar `F8`.

## Como rodar

```powershell
pip install -r requirements.txt
python -m playwright install firefox
python autocliquer.py
```

No app, escolha Brave, Firefox ou Chrome no campo de navegador. O Brave e o Chrome precisam estar instalados no Windows.
